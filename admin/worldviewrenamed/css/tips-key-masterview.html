Tips
  • • Records can be navigated and edited inline;just type in the field and "enter", "tab" or "right-arrow" to save changes and/or navigate the fields.
  • •To edit a row using the 'edit' button, click the checkbox of the row you want edit, and then click the edit button.
  • •You can update fields in multiple records at once,(i.e. status and substatus changes, or updating run date and lane number) by holding down the control key while clicking row checkboxes and then click the edit button.(Only update fields with identical values)
  • •Columns are sortable using the arrows at the right side of the column label. Columns can be dragged to new positions. If you specify a preferred view for any record set, please let me know and I will create a custom view for your convenience
